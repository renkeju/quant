# NumPy 高级索引

NumPy 比一般的Python序列提供更多的索引方式。除了之前看到的用整数和切片的索引外，数组可以由整数数组索引，布尔索引及花式索引。

## 整数数组索引

以下实例获取数组中的(0,0),(1,1)和(2,0)位置处的元素。

```
>>> import numpy as np
>>> x = np.array([[1, 2], [3, 4], [5, 6]])
>>> y = x[[0, 1, 2], [0, 1, 0]]
>>> print(y)
[1 4 5]
```

以下实例获取了 4*3 数组中的四角的元素。行索引是`[0,0]`和`[3,3]`，而列索引是`[0,2]`和`[0,2]`。
