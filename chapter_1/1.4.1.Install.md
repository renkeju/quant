# 安装

我们使用 pip 工具来安装 Matplotlib 库。

安装 matplotlib 库：

```
pip3 isntall matplotlib
```

安装完成后，我们就可以通过 import 来导入 matplotlib 库：

```python
import matplotlib
```

以下实例，我们通过导入 matplotlib 库，然后查看 matplotlib 库的版本号：

```python
import matplotlib

print(matplotlib.__version__)
```

执行以上代码，输出结果如下：

```
3.4.2
```
